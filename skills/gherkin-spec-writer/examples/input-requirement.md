# Example Input — gherkin-spec-writer

## User Story

```
US-042: Money Transfer

As a mobile application user,
I want to transfer money to another registered user,
So that I can send funds instantly and securely.

Acceptance Criteria:
- The user must be authenticated
- Amount must be between 1 and 10,000 USD
- The recipient must have an active account
- The user must have sufficient balance
- Both sender and recipient receive a notification on success
- A transaction record is created and visible in the history
```
